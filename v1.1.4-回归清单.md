## 2. 功能回归清单

### 2.1 IME 与 C 取色冲突

- [ ] 中文输入法开启时，放大镜可见 + 按 `C` 能复制色值
- [ ] 英文输入状态下，放大镜可见 + 按 `C` 能复制色值
- [ ] 文字标注输入中（焦点在文本框）按 `C` 不触发全局复制
- [ ] 放大镜不可见时按 `C` 不触发色值复制

### 2.2 剪贴板三模式

- [ ] 仅图片模式：连续 20 次截图写入成功
- [ ] 图片 + 路径模式：连续 20 次写入成功
- [ ] 仅文本模式：连续 20 次写入成功
- [ ] 截图完成到提示出现期间，UI 无明显停顿

### 2.3 马赛克轨迹采样

- [ ] 连续涂抹 60 秒，轨迹连续无明显断裂
- [ ] 收笔时末点位置正确，无尾端缺口
- [ ] 长时间涂抹后内存占用增长受控（非线性暴涨）

### 2.4 马赛克背景按需生成

- [ ] 未切换马赛克工具时，流程中不生成马赛克背景
- [ ] 首次切换马赛克工具时才触发背景生成
- [ ] 选区变更后再次使用马赛克，图像对齐无偏移/错位

### 2.5 日志异步化与跨天切换

- [ ] 高频操作下 UI 无明显卡顿
- [ ] 日志写入正常（无丢失、无异常中断）
- [ ] 跨天后新日志写入新日期文件
