# 更新日志

All notable changes to this project will be documented in this file.

## [v1.2.1] - 2026-03-18

> **[SYS_MSG]** 增强 Markdown/HTML 路径复制的稳定性与输出策略，并将托盘菜单升级为“最近一次 + 快速切换”的效率入口；同时修复设置页打开崩溃与禁用态视觉不明显的问题。
> **[METRICS]** Binary: <1MB

### 💻 终端与工作流
* **[工作流]** README 同步更新：版本徽章、托盘“最近一次/快速切换”说明与 Markdown/HTML 输出示例。

### ✨ 新特性
* **[UX]** 托盘菜单升级为“最近一次 + 快速切换” - 支持复制最近一次图片/路径、打开文件/资源管理器定位，并可快速切换剪贴板模式与路径格式。
* **[Core]** Markdown/HTML 路径复制增强 - 默认输出 `file:///...` 的 AbsoluteUri，并支持“仅片段/仅纯路径/片段+纯路径”策略。
* **[UX]** 托盘复制最近一次路径支持临时输出格式（纯文本/Markdown/HTML），仅影响本次复制，不修改全局配置。
* **[UX]** 设置页根据剪贴板模式动态置灰无效选项，并展示提示文案，减少“可选但不生效”的误解。

### 🐛 问题修复
* `[UI]` **修复打开设置时崩溃** - 初始化阶段触发 `SelectionChanged` 时增加保护与空值兜底，避免空引用异常导致闪退。
* `[UI]` **修复托盘交互误导** - 非仅路径模式下禁用“路径格式”菜单，避免可选但不生效。
* `[UI]` **修复禁用下拉框不灰显** - 自定义 `ComboBox` 模板补齐 `IsEnabled=False` 视觉状态。

### ⚙️ 优化与重构
* `[Core]` **图片+路径模式文本语义收口** - 随图附带文本固定为 Windows 纯路径，避免 Markdown/HTML 片段污染。
